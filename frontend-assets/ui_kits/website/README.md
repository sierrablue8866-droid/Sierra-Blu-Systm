# Meridian Website UI Kit

A high-fidelity click-through prototype of the Meridian Properties real estate website.

## Components

- `Header.jsx` — Frosted-glass navigation bar, wordmark, links, CTA
- `HeroSection.jsx` — Full-bleed cinematic hero with headline and search bar
- `SearchBar.jsx` — Compound search input with filter chips
- `PropertyCard.jsx` — Listing card with image, specs, badge, hover states
- `ListingsGrid.jsx` — Masonry/grid layout of property cards
- `PropertyDetail.jsx` — Full property detail page (gallery, specs, agent, contact)
- `Footer.jsx` — Minimal footer with links, wordmark, legal

## Usage

Open `index.html` to see the full interactive prototype. Click any property card to navigate to the detail view. Use the nav links to switch between Buy / Rent / Sell views.

## Design width

1440px desktop, responsive down to 1024px.
